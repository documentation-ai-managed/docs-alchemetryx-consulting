---
title: Workflow Sync Issues
description: Fix Alchemetryx dashboard sync issues.
---

## Symptoms
- Data on your dashboard is outdated
- Workflow status doesn't match recent actions

## Causes
- Integration with source apps is disconnected
- Manual sync hasn't run

## Quick Fix
<Steps>
  <Step title="Check integration status">
    ````
    Visit Integrations in dashboard. Ensure all are 'Connected.'
    ````
  </Step>
  <Step title="Run manual sync">
    ````
    Click 'Sync Now' on your dashboard to refresh data instantly.
    ````
  </Step>
</Steps>

<Callout kind="tip">
If issues persist, contact support for troubleshooting.
</Callout>
